# Weather_Analysis
This project is an end-to-end data pipeline that collects real-time weather and air quality data for major districts in Tamil Nadu using the OpenWeatherMap API. The data is processed using Python, stored in a MySQL database, and exported as a CSV file for analysis and visualization.
